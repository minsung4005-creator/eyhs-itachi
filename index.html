WEB 언양고 오늘의 급식봇

🔍 프로젝트 개요

매일 아침 학교 홈페이지나 복도에 부착된 급식표를 찾으러 가야 하는 불편함에서 시작되었습니다. 동아리 세미나에서 웹 크롤링과 API 통신을 배운 후, 교육부 NEIS Open API를 활용해 학급 친구들이 스마트폰 홈 화면에서 위젯처럼 쉽게 학교 메뉴를 읽을 수 있도록 만들었습니다.

🛠 핵심 기술 & 구현 방식

  • Fetch API를 사용한 비동기 데이터 통신.
  • 언양고 고유 학교코드(7480112)와 교육청 코드(M10) 파라미터 매핑.
  • 정규식을 활용해 급식 원재료에 섞인 알레르기 유발 물질 번호 자동 필터링 및 서식 가공.
🔥 배운 점 & 느낀 점

처음으로 실제 배포되어 친구들이 쓰는 모습을 보았습니다. UI 편의성이 성능만큼이나 중요하다는 것을 배웠으며, 서버 점검 등으로 API 응답이 없을 때를 대비한 예외 처리(Fallback) 로직의 필요성을 절실히 느꼈습니다.

GAME 닷지마스터 (Dodge Master)

🎮 프로젝트 개요

파이썬의 절차지향 코딩에서 객체지향 프로그래밍(OOP)으로 실력을 한 단계 성장시키기 위해 제작한 프로젝트입니다. 동아리 학술제 전시용 게임부스로 출품되어 전교생의 뜨거운 신기록 경쟁을 불러일으켰습니다.

🛠 핵심 기술 & 구현 방식

  • 플레이어, 방해물, 아이템 객체(Class) 설계 및 캡슐화 구현.
  • `Rect.colliderect`를 활용한 박스 형태 충돌 보정 시스템 구축.
  • 시간 경과에 따른 난이도(스폰율, 장애물 속도) 상승 알고리즘 설계.
👾 배운 점 & 느낀 점

동아리 부원들이 게임을 무조건 부수려고 하는 것(버그 테스팅)을 보고, 안정적인 코드 작성이 가장 기본이 되어야 함을 깨달았습니다. 클래스 분할 설계를 통해 추후 새로운 장애물을 추가할 때 기존 코드를 수정하지 않고 쉽게 유연하게 확장할 수 있는 OOP의 위대함을 알게 되었습니다.

IoT 아두이노 스마트 온실 시스템

🌿 프로젝트 개요

동아리실에 있는 화분들이 방학 기간 동안 시들어 죽는 것을 보며 안타까운 마음에 기획했습니다. 소프트웨어 개발 기술과 하드웨어를 접목하여 물리 세상에 직접 개입하고 제어해보는 색다른 재미를 느낄 수 있었습니다.

🛠 핵심 기술 & 구현 방식

  • DHT11 온습도 센서와 토양 수분 센서 데이터 융합 처리.
  • 5V 워터 펌프 모터 및 모터 드라이버(L298N) 제어 제어.
  • 16x2 I2C LCD 스크린을 통한 실시간 현재 상태 시각화 출력.
⚡ 배운 점 & 느낀 점

현실 세계의 센서값은 예상보다 노이즈가 많다는 것을 알게 되었으며, 수분 센서 값이 튀는 현상을 소프트웨어 평균 필터 알고리즘을 만들어 방어해냈습니다. 이 과정에서 순수 개발을 넘어 하드웨어 인터페이스와 프로토콜에 대한 이해를 크게 기를 수 있었습니다.
